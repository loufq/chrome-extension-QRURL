chrome-extension-QRURL
======================

chrome-extension-QRURL build on YO 

[QRURL-on chrome google.com](https://chrome.google.com/webstore/detail/qrurl/kkemaebdacbokdnlmaakingnfpccpcil?hl=zh-CN)


这是用[YO-generator-chrome-extension](https://github.com/yeoman/generator-chrome-extension)构建的APP

有了 yeoman[YO]，你只要写代码就够了（在有YO之前，开发chrome-extension app,基本上花在非写代码时间上占整个应用的40%时间左右吧，有了YO后基本只要写代码就可以）

[generator-chrome-extension](https://github.com/yeoman/generator-chrome-extension) 所见即所得，修改即更新，类是nodejs下的nodemon

开发调试只要 grunt debug

创建发布包只要 grunt build
